Focusing on HTML

HTML is not a programming language, rather, it is a markup language. This is a tool used to define the structure of a web page.

It is not case sensetive, although typically HTML will be written in lowercase.

This is the foundation of all web pages, and can be used in conjunction with javascript to creative interactive web pages.

HTML has the power to nest elements inside of itself.

HTML has been managed by the World Wide Web Consortium since 1996, but the Web Hypertext Application Technology Working Group become the only publisher of HTML during 2019.
