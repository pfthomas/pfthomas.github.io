HyperText

Markup Language

HTML stands for HyperText Markup Language. This is a text based language that is a fundamental building block of modern webpages. The syntax and concept is a plaintext document structured with elements. Elements are surrounded by matching opening and closing tags. Each tag begins and ends with angle brackets (<>).
